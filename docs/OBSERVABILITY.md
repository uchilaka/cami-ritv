# CAMI: Observability

## Logs

Logs are a critical component of observability in production environments. They provide insights into the behavior of applications and systems, allowing developers and operators to identify and troubleshoot issues effectively.

### Configuration: `production`

> Review [this guide](https://betterstack.com/docs/logs/render/?source=1734918) for the steps to set up log forwarding from Render to Better Stack.

The production logs are forwarded to Better Stack from Render. To access the logs, you can use the Better Stack dashboard.
